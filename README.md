## Food Festival PWA :

A Progressive Web App for browsing food festival events, schedules, and ticket purchasing.
Features

    📱 Installable on mobile devices

    🔄 Offline support with service worker

    📅 Event schedule viewer

    🎫 Ticket purchase interface

## Tech Stack :

    HTML5/CSS3

    JavaScript

    Webpack

    Service Worker API

    Manifest.json for PWA

## Quick Start :


# Install dependencies
npm install

# Build the project
npm run build

# Start development server
npm start

## PWA Implementation :

    Service worker caches HTML, CSS, JS, and assets

    Manifest provides installable app experience

    Offline fallback support
